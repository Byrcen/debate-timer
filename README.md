# 辩论赛计时器 · Pro

一个单文件、零依赖的辩论赛主持计时网页。双击 `主持文件.html` 即可使用，无需安装任何东西。

红方（正）与蓝方（反）各拥有一团火焰：谁发言，谁的半边屏幕燃起火来；剩余时间越少，火烧得越高越旺，最后几秒数字着火、全屏火墙，时间到的瞬间火星伴随法槌轰然炸开。

## 赛制规则

内置五人制赛制，共 18 个环节（各环节时长可在「设置 → 环节时长」中调整）：

| # | 阶段 | 环节 | 时长 | 说明 |
|---|------|------|------|------|
| 01 | 开场 | 双方团队自我介绍 | 不计时 | 双方 1-5 辩依次自我介绍 |
| 02 | 开篇明义 | 正方一辩 · 开篇立论 | 3:40 | 陈述本方论点及论据 |
| 03 | 开篇明义 | 反方五辩 · 质疑正方一辩 | 1:30 | 回答方享 5 秒保护，5 秒后方可打断；回答方只答不反问 |
| 04 | 开篇明义 | 反方一辩 · 开篇立论 | 3:40 | 陈述本方论点及论据 |
| 05 | 开篇明义 | 正方五辩/四辩 · 质疑反方一辩 | 1:30 | 规则同上 |
| 06 | 申论环节 | 正方二辩 · 申论 | 2:00 | 申论、反驳或申论 + 反驳 |
| 07 | 申论环节 | 反方二辩 · 申论 | 2:00 | 同上 |
| 08 | 申论环节 | 对辩环节 · 双方二辩 | 每方 1:30 | 单边计时，一方停止发言后另一方才可发言 |
| 09 | 盘问环节 | 反方三辩 · 盘问 | 1:30 | 盘问对方除三辩外任意选手，回答方享 5 秒保护 |
| 10 | 盘问环节 | 反方 · 盘问小结 | 1:00 | 反方三辩总结盘问过程 |
| 11 | 盘问环节 | 正方三辩 · 盘问 | 1:30 | 规则同上 |
| 12 | 盘问环节 | 正方 · 盘问小结 | 1:00 | 正方三辩总结盘问过程 |
| 13 | 补充说明 | 正方四辩 · 补充说明 | 1:00 | 针对质疑环节未解决的问题补充 |
| 14 | 补充说明 | 反方四辩 · 补充说明 | 1:00 | 同上 |
| 15 | 自由辩论 | 自由辩环节 | 每方 4:00 | 单边计时，轮换发言，同队不得连续两人发言，正方先开始 |
| 16 | 总结陈述 | 反方五辩 · 总结陈述 | 3:00 | 总结对辩、申论本方论点并升华 |
| 17 | 总结陈述 | 正方五辩 · 总结陈述 | 3:00 | 同上 |
| 18 | 赛后 | 评委点评 & 最佳辩手 | 不计时 | 评委点评双方表现，各队内评最佳辩手 |

**质询/盘问保护规则**：质询环节按 <kbd>Q</kbd> 进入"回答模式"——主计时冻结，回答方享 5 秒不可打断保护（护盾音效起，水晶铃止）；回答结束再按 <kbd>Q</kbd> 恢复质询计时。

**超时规则**：单人环节时间到不停表，红色 `+MM:SS` 继续正计时并记入赛后报告；对辩/自由辩用完即冻结，不可再发言。

## 主持功能

- **计时核心**：真实时间戳驱动，切标签页、共享屏幕不丢秒；3-2-1 全屏入场倒计时；最后 30 秒 / 3-2-1 音效提醒；时间到法槌 + 冲击波
- **防误操作**：计时中切环节需按两次确认；<kbd>Z</kbd> 一键撤销；关闭页面前浏览器确认；计时中屏幕保持常亮
- **断线恢复**：比赛进度实时保存，刷新 / 崩溃后一键恢复到原环节原秒数
- **赛后报告**：自动记录各环节实际用时、超时时长、对辩逐次发言记录，一键复制或下载 .txt
- **赛事设置**：比赛标题、辩题、场次、队名、音效音量、各环节时长均可自定义（本地保存）
- **全程音效**：Web Audio 实时合成，封面「♪ 音效」页可逐一试听

## 快捷键

| 键 | 功能 |
|----|------|
| <kbd>Space</kbd> | 开始 / 暂停（环节从零开始时先播 3-2-1 倒计时） |
| <kbd>←</kbd> <kbd>→</kbd> | 上一个 / 下一个环节 |
| <kbd>S</kbd> | 对辩 / 自由辩切换发言方 |
| <kbd>Q</kbd> | 质询环节：回答开始 / 结束 |
| <kbd>R</kbd> | 重置当前环节 |
| <kbd>Z</kbd> | 撤销上一次切环节 / 重置 |
| <kbd>M</kbd> | 静音 / 取消静音 |
| <kbd>F</kbd> | 全屏 |

## 使用

1. 下载 `主持文件.html`，用 Chrome / Edge / Safari 打开
2. 点「⚙ 设置」填写比赛标题、辩题、队名
3. 点「开始比赛」，跟着流程走即可；左上 ☰ 可打开完整流程列表跳转
4. 赛后点「📋 报告」导出用时统计
